## 1. 引言

### 1.1 编写目的

本文档旨在明确 Subtitle Translator Local 产品的功能需求、非功能需求及约束条件，为产品设计、开发、测试及验收提供依据。目标读者包括产品经理、开发工程师、测试工程师及项目相关人员。

### 1.2 产品概述

Subtitle Translator Local 是一款基于本地大语言模型的批量字幕翻译工具。产品通过 Ollama API 接口调用本地部署的 AI 大模型，实现字幕文件的自动化翻译，满足视频内容国际化需求。

**核心价值主张**：
- **隐私保护**：完全本地化处理，无需上传数据至云端
- **零运营成本**：无需支付第三方 API 调用费用
- **高质量翻译**：结合上下文的智能翻译，准确传达原文语义

### 1.3 术语定义

| 术语 | 定义 |
|------|------|
| SRT | SubRip 字幕格式，一种广泛使用的字幕文件格式 |
| Ollama | 本地大语言模型运行平台，支持多种开源模型 |
| 批处理 | 将多个字幕条目合并为单次 API 请求的处理方式 |
| 预热 | 在正式翻译前预先加载模型到内存的操作 |

### 1.4 参考资料

- Ollama 官方文档: https://ollama.ai/
- SubRip 字幕格式规范
- GB/T 8567-2006 计算机软件文档编制规范

### 2.3 产品边界

**包含范围**：
- SRT 格式字幕文件的翻译处理
- Ollama 平台兼容模型的调用
- 基础翻译配置管理

**不包含范围**：
- 其他字幕格式（ASS/SSA/VTT）的直接支持
- 在线翻译 API 服务集成
- 字幕时间轴编辑功能
